#15 - داشبورد مخصصة لكل دور
What & Why
الداشبورد الحالية تعرض نفس المحتوى لكل الأدوار تقريباً مع فروق بسيطة. المطلوب إعادة تصميم الداشبورد بحيث كل دور يرى نسخة مختلفة تماماً تناسب شغلانته، على غرار شركة بروكر عقاري مصري حقيقية.

Done looks like
سيلز (sales_agent): يرى فقط أداءه الشخصي — ليدزه، نسبة تحويله، عمولاته، تذكيراته القادمة، حالات ليدزه، و Lead Score (هوت / وورم / كولد) الخاصة به.
تيم ليدر (team_leader): يرى أداء تيمه — توزيع الليدز على أفراد التيم، مين شغال ومين لأ، الليدز الهوت اللي محتاجة متابعة عاجلة في التيم، عمولات التيم، ونسبة تحويل التيم.
سيلز ادمن / سيلز مانجر (sales_admin / sales_manager): يرى الصورة التشغيلية الكاملة — الليدز الغير موزعة (unassigned)، load balancing بين التيمات، معدل response time، أداء كل التيمات، تقرير يومي/أسبوعي.
صاحب الشركة (company_owner): يرى الصورة التجارية الكبيرة — إجمالي العمولات والإيرادات، نسبة التحويل الكلية، مصادر الليدز الأفضل أداءً، مقارنة التيمات، أفضل سيلز في الشهر.
مدير النظام (super_admin / admin): يرى كل حاجة — نفس صاحب الشركة + عدد المستخدمين النشطين وإحصائيات عامة على النظام.
Out of scope
تغيير الـ API أو صلاحيات الوصول للبيانات (دي موجودة وشغالة)
تغيير الـ sidebar navigation
إضافة charts/graphs جديدة معقدة — الـ Recharts PieChart الموجود يكفي
Tasks
تعريف نسخة الداشبورد لكل دور — إعادة هيكلة dashboard.tsx باستخدام role كـ switch/condition رئيسي يحدد أي layout يتعرض، بدل الـ conditionals المبعثرة الحالية.

داشبورد السيلز — تصميم نسخة مخصصة لـ sales_agent تتضمن: KPIs شخصية (ليدزه / نسبة تحويله / عمولته)، Lead Score Summary (هوت/وورم/كولد من ليدزه بس)، توزيع حالات ليدزه، والتذكيرات القادمة.

داشبورد التيم ليدر — تصميم نسخة لـ team_leader تتضمن: أداء كل سيلز في تيمه (جدول بالاسم / عدد الليدز / نسبة التحويل)، الليدز الهوت في التيم، عمولات التيم، وKPIs خاصة بالتيم.

داشبورد السيلز ادمن/مانجر — تصميم نسخة لـ sales_admin و sales_manager تتضمن: عدد الليدز الغير موزعة مع زر سريع للتوزيع، load balancing chart بين السيلزية، معدل response time، وتقرير أداء يومي.

داشبورد صاحب الشركة ومدير النظام — تصميم نسخة علوية لـ company_owner و super_admin/admin تتضمن: إجمالي العمولات الشهرية، نسبة التحويل الكلية، مقارنة التيمات، أفضل سيلز، وعدد المستخدمين النشطين (للـ admin فقط).
