# PRD - HomeAdvisor CRM
## Product Requirements Document
### مقارنة مع ProfitCRM وخطة التطوير

## 1. نظرة عامة

### 1.1 الهدف
بناء نظام CRM متكامل لإدارة العملاء المحتملين (Leads) في قطاع العقارات، منافس لـ ProfitCRM مع دعم كامل للعربية والإنجليزية.

### 1.2 الجمهور المستهدف
- شركات التطوير العقاري
- الوسطاء العقاريون (Brokers)
- وكالات التسويق العقاري
- فرق المبيعات

## 7. قاعدة البيانات - الجداول المطلوبة

### جداول جديدة مقترحة:

```sql
-- الحملات التسويقية
CREATE TABLE campaigns (
  id VARCHAR PRIMARY KEY,
  name TEXT NOT NULL,
  channel TEXT,
  start_date TIMESTAMP,
  end_date TIMESTAMP,
  budget DECIMAL,
  status TEXT DEFAULT 'active',
  created_at TIMESTAMP DEFAULT NOW()
);

-- التذكيرات
CREATE TABLE reminders (
  id VARCHAR PRIMARY KEY,
  lead_id VARCHAR REFERENCES leads(id),
  user_id VARCHAR REFERENCES users(id),
  title TEXT NOT NULL,
  description TEXT,
  due_date TIMESTAMP NOT NULL,
  is_completed BOOLEAN DEFAULT false,
  created_at TIMESTAMP DEFAULT NOW()
);

-- المدفوعات
CREATE TABLE payments (
  id VARCHAR PRIMARY KEY,
  lead_id VARCHAR REFERENCES leads(id),
  unit_id VARCHAR REFERENCES units(id),
  amount DECIMAL NOT NULL,
  payment_type TEXT,
  payment_date TIMESTAMP,
  notes TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);

-- الوسطاء
CREATE TABLE brokers (
  id VARCHAR PRIMARY KEY,
  name TEXT NOT NULL,
  phone TEXT,
  email TEXT,
  company TEXT,
  commission_rate DECIMAL,
  is_active BOOLEAN DEFAULT true,
  created_at TIMESTAMP DEFAULT NOW()
);

-- صور المشاريع
CREATE TABLE project_images (
  id VARCHAR PRIMARY KEY,
  project_id VARCHAR REFERENCES projects(id),
  image_url TEXT NOT NULL,
  is_primary BOOLEAN DEFAULT false,
  created_at TIMESTAMP DEFAULT NOW()
);

-- الإشعارات
CREATE TABLE notifications (
  id VARCHAR PRIMARY KEY,
  user_id VARCHAR REFERENCES users(id),
  title TEXT NOT NULL,
  message TEXT,
  type TEXT,
  is_read BOOLEAN DEFAULT false,
  created_at TIMESTAMP DEFAULT NOW()
);
```
